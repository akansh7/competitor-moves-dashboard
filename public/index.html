API Reference

POST /api/report

Push a new competitor intelligence report to the dashboard.

{
  "strategic_headline": "string",
  "what_they_are_building": "string",
  "team_growth_signals": "string",
  "competitive_threat_level": "High | Medium | Low",
  "threat_rationale": "string",
  "recommended_action": "string",
  "key_skills_emerging": ["string"],
  "top_job_titles": ["string"]
}

GET /api/reports

Retrieve all stored reports as JSON.

DELETE /api/reports

Clear all stored reports.

WebSocket

Connect to ws://localhost:3000 to receive live updates.

Example cURL
curl -X POST http://localhost:3000/api/report \
  -H "Content-Type: application/json" \
  -d '{
    "strategic_headline": "Adobe is embedding GenAI...",
    "what_they_are_building": "They are investing in...",
    "team_growth_signals": "One mid-level position...",
    "competitive_threat_level": "Medium",
    "threat_rationale": "While hiring volume is low...",
    "recommended_action": "Our GTM team should...",
    "key_skills_emerging": ["iOS Development", "Swift", "GenAI"],
    "top_job_titles": ["Senior iOS Developer - GenAI"]
  }'
